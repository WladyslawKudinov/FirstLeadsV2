# Cross-Cutting Patterns

When brainstorming, notice these repeat across steps — one vehicle can kill multiple problems:

**"Scattered information"** (Steps 7, 11, 16) — Interview-based extraction, "send whatever you have," etc.

**"Can't evaluate / never seen this"** (Steps 12, 13, 14, 15) — One guided review framework solves 4 steps.

**"No time / keep postponing"** (Steps 12, 13, 14, 15, 16) — Any vehicle capping effort at 10-15 minutes.

**"Why are you asking me?"** (Steps 12, 13, 19) — Clear upfront map of which 5 decisions need the CPO.

**"Confirmation bias / can't interpret"** (Steps 22, 23, 24) — Built-in analysis the CPO receives as recommendations.
