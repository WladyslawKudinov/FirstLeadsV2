# Research Brief #2 — Поиск конкретных компаний

**Продукт:** AI-ассистент для бизнеса (RAGProduct)
**Дата:** 20 февраля 2026
**Цель:** Найти 7-10 конкретных компаний в каждом из Tier 1-2 сегментов, подходящих для первых продаж

---

## Контекст для исследователя

**Что ищем:** Конкретные компании для первых 5-8 продаж AI-ассистента для бизнеса.

**Продукт (кратко):** AI-ассистент + методология внедрения + обучение сотрудников. Экономит время на рутине: генерация КП, ответы на претензии, поиск по внутренним документам, суммаризация. Цена: проект ~750K₽ + подписка 80-110K₽/мес.

**Что искать в каждой компании:**
1. Название, город, выручка, количество сотрудников
2. Почему подходят (конкретные сигналы)
3. Сигналы боли (вакансии, тендеры, новости, обсуждения)
4. Возможный ЛПР (должность, имя если доступно)
5. Точка входа (как дотянуться)

### Где искать компании

**Rusprofile.ru / SPARK:**
- Фильтр: ОКВЭД 46.x, выручка 100-500M₽, сотрудники 30-200, Москва/СПб/регионы
- Приоритет: стабильные (5+ лет), растущие (выручка YoY+)

**HH.ru — сигналы боли:**
- Поиск: «менеджер по продажам» + «оптовая торговля» + «дистрибьютор»
- Что искать в вакансиях: описание рутинных задач (КП, переписка, претензии), упоминание CRM, количество вакансий (если 3+ одновременно → высокая текучка → боль)
- Вакансии «коммерческий директор» / «РОП» с описанием «внедрение автоматизации» → сигнал готовности

**LinkedIn:**
- Поиск: «коммерческий директор» / «руководитель отдела продаж» в торговых компаниях
- Активность: посты про AI, автоматизацию, CRM → сигнал интереса

**Telegram:**
- Каналы: @Grebenukresulting, @salesnotes, @dirclub — кто из подписчиков является ЛПР в торговых компаниях?
- Обсуждения: жалобы на рутину, вопросы про AI для продаж

**Zakupki.gov.ru (для компаний с госзаказчиками):**
- Торговые дома, которые участвуют в тендерах → подтверждение масштаба, формализованные БП

### Приоритетные подвертикали

1. **Дистрибьюторы стройматериалов** — большие каталоги, сложные КП, много SKU
2. **Дистрибьюторы электроники / компонентов** — техническая переписка, ответы на тех. вопросы
3. **Оптовые продовольственные компании** — высокая частота заказов, много типовых документов
4. **Дистрибьюторы медоборудования / фармацевтики** — регуляторика, много регламентов

- Дополнительный фильтр: наличие слов «конструктор» / «технолог» в вакансиях на HH.ru → подтверждает инженерный офис

**HH.ru — сигналы боли:**
- Поиск: «инженер-конструктор» / «инженер-технолог» + конкретная компания

**isicad.ru / sapr.ru:**
- Кейсы внедрения САПР в российских компаниях → список компаний с инженерными офисами
- Форумы: обсуждения проблем с документацией

**Закупки / тендеры:**
- zakupki.gov.ru: тендеры на PDM/PLM среди производственных компаний → кто ищет решение для документации
- Тендеры на САПР-лицензии → подтверждает наличие инженеров

### Приоритетные подвертикали

1. **Машиностроение / приборостроение** — максимум техдокументации, ЕСКД, сложные изделия
2. **Производство металлоконструкций** — проектная документация, спецификации
3. **Пищевое производство** — регламенты, ТУ, ХАССП-документация
4. **Производство электроники** — BOM-листы, спецификации компонентов
