# Research Brief #1 — AI-ассистент для бизнеса (RAGProduct)

**Дата:** 20 февраля 2026
**Цель:** Собрать данные для скоринга 8 сегментных гипотез по Moore Beachhead Criteria
**Формат результата:** По каждому сегменту — заполненные разделы ниже. Где данные не найдены — писать "NOT FOUND" (не пропускать).

## Контекст для исследователя

**Что за продукт:**
AI-ассистент для бизнеса — это не просто RAG-система для поиска по документам, а полноценный рабочий инструмент + методология внедрения + обучение сотрудников. Продукт:
- Готовит ответы на письма клиентам/партнёрам
- Анализирует записи переговоров и выделяет ключевое
- Готовит ответы на претензии с учётом внутренних регламентов
- Даёт справки по любому вопросу — ищет и в интернете, и во внутренних документах
- Суммаризирует длинные документы и переписки

**Ключевое отличие:** Не софт, а методология внедрения AI в бизнес-процессы. 4 этапа: анализ БП → карта внедрения → внедрение инструментов → поддержка. Плюс обучение сотрудников (без обучения AI-инструменты не используют).

**Целевой клиент:** Компании 30-300 сотрудников, выручка 100-1000М₽, у которых есть бизнес-процесс с рутинными задачами (письма, справки, ответы, документы). Хотят AI, но не знают как начать, боятся утечки данных.

**Ценовая модель:**
- Демонстрация: бесплатно
- Проект внедрения: от 600,000 ₽ (средний ~750,000 ₽)
- Подписка: 80,000-110,000 ₽/мес (включает хостинг, API, поддержку до 10 ч/мес)

**Рынок:** Россия. AI-интеграция для среднего бизнеса — рынок практически пустой. Нет устоявшихся игроков.

**Конкуренты:**
- Публичный ChatGPT/Claude (данные уходят наружу)
- «Сами разберёмся» (80% провал)
- Just AI (другой фокус — чат-боты для крупного бизнеса)
- Microsoft Copilot (дорого, не для РФ, нет обучения)

## Deliverable Checklist

По каждому сегменту должны быть заполнены:

| Раздел | Минимальный результат |
|--------|----------------------|
