Убитый: 5.1 VC-backed B2B tech (Series A-B)

  1. Размер слишком мал. 24 публичные B2B Software сделки за 2024. ~40-80 компаний — не сегмент, а список имён.
  2. Неверный timing. Series A-B уже имеют paying customers. Им нужно масштабирование, не валидация.
  3. Рынок сжат. $91-177M за весь 2024. Ставка ЦБ 21%. Активность минимальна.

Применение: PR/reference-канал, не целевой сегмент.

Критическое ограничение: скорость

Наблюдения для review

  1. A и B — одни компании, разные триггеры. Возможно один мега-сегмент с двумя messaging tracks. Решение отложено.
  2. C требует отдельного GTM. Другой ЛПР, messaging, ценник. Фактически второй продукт.
  3. Observability решена для A и B. Рейтинги + HH.ru = список за 30 минут. Для C — через Сколково/ФРИИ. Для D — сложнее.
  4. Pricing signal. Custdev 33K-550K руб. UXSSR 1-3M руб. FL 150-350K = «средний сегмент агентств».
  5. 5.1 (VC-backed) полезен как PR-канал. Кейс = reference для всех остальных.
