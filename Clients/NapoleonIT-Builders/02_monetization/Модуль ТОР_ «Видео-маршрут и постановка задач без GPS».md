## **Модуль ТОР: «Видео-маршрут и постановка задач без GPS»**

---

## **1\. Назначение модуля**

Модуль предназначен для:

* фиксации маршрута обхода сотрудника **без использования GPS**

* привязки маршрута к видеообходу

* автоматической или ручной постановки задач

* формирования отчётности по прохождению регламентов

* аналитики качества исполнения

Решение применяется на строительных объектах, в эксплуатации, промышленности, ритейле и на режимных объектах, где использование GPS ограничено или запрещено.

---

## **2\. Принцип формирования стоимости**

Стоимость тарифа зависит от объективной нагрузки на систему и бизнес-ценности для заказчика.

### **Ключевые параметры ценообразования:**

1. Объём видеоанализа (часов в месяц)

2. Количество пользователей

3. Период хранения видео

4. Уровень AI-аналитики

5. Модель размещения (Cloud / On-Prem)

Основным драйвером является **объём видеоанализа**, так как именно он формирует нагрузку на вычислительные мощности и хранение.

# **4\. Логика разбега стоимости (25–40 тыс ₽)**

Разбег обусловлен:

| Фактор | Влияние |
| ----- | ----- |
| Объём видео | Рост вычислительной нагрузки |
| Длина маршрутов | Увеличение AI-обработки |
| Количество смен | Увеличение часов анализа |
| Срок хранения | Рост расходов на облако |
| Глубина аналитики | Использование GPU |

Таким образом:

* Малый объект (30–40 часов видео) → экономически оправдан тариф 25 000 ₽

* Средний объект (70–100 часов) → 32 000 ₽

* Крупный объект (150+ часов) → 40 000 ₽ и выше

Это позволяет сохранять маржинальность при масштабировании.

# **6\. Сравнение Cloud vs On-Prem**

| Параметр | Cloud | On-Prem |
| ----- | ----- | ----- |
| Стартовые затраты | Низкие | Высокие |
| Регулярные платежи | Подписка | Поддержка |
| Масштабируемость | Высокая | Ограничена сервером |
| Скорость внедрения | 1–2 недели | 1–3 месяца |
| Капитализация бизнеса | Высокий MRR | Разовые сделки |
