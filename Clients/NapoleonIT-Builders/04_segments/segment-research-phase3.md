## Bowling Pin Strategy (Final)

```
BEACHHEAD                  FAST FOLLOW                    EXPANSION
─────────                  ───────────                    ─────────

🥇 Стройконтроль ──────→ 🥇 Приёмка квартир ──────→ 🥈 Нефтегаз
   генподрядчиков            (тот же девелопер,           (после решения
   (АПРИ pilot ✓)             другой отдел)                C3-blocker)

   WHY FIRST:               WHY SECOND:                  WHY THIRD:
   • Есть пилот              • Голубой океан              • Крупные чеки
   • Есть founder access     • Кросс-продажа             • Престижный референс
   • Каналы известны          • Триггер: мораторий        • Требует on-prem
```

### Bowling Pin Logic — Детали

**Pin 1 → Pin 2:** Один и тот же девелопер (напр. DOGMA) покупает ТОР для стройконтроля → видит, что технология работает → предлагает отделу качества для приёмки квартир. **Продажа внутри аккаунта, не cold outreach.**

**Pin 2 → Pin 3:** Референсы «крупнейший девелопер России использует ТОР» → нефтегаз видит зрелость продукта. Параллельно решается C3 (on-prem для КИИ).
