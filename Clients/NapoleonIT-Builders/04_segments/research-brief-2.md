# Research Brief #2: Company-Level Research

**Продукт:** Система «ТОР» (Napoleon IT)
**Цель:** Найти конкретные компании для outreach в каждом приоритетном сегменте
**Дата:** 20 февраля 2026

---

## Context for Researcher

**Что за продукт:** CV-система для контроля обходов на строительных объектах. Фиксирует траекторию инспектора на плане + привязывает к видео. Работает без маяков/меток — только камера.

**Что мы выяснили в Phase 2:**

| Сегмент | Приоритет | Почему |
|---------|-----------|--------|
| **Девелоперы — Приёмка квартир** | 🥇 Beachhead | Голубой океан (нет видео-решений), 211k исков/год, 128 млрд ₽ убытков |
| **Стройконтроль генподрядчиков** | 🥇 Beachhead | Есть пилот АПРИ, OpenSpace ушёл, каналы известны |
| **Нефтегаз** | 🥈 Fast Follow | Крупные бюджеты, но требует on-prem + КИИ |

**Что нужно от исследования:** Конкретные компании для outreach + точки входа (события, сообщества, интеграторы).

---

## Segment 1: Девелоперы — Приёмка квартир

### Company Profile

**Кого ищем:**
- Застройщики жилья из топ-50 по ЕРЗ.РФ
- Сдача 1000+ квартир/год
- Регион: Москва/МО (50-55% споров), СПб, Екатеринбург, Новосибирск, Краснодар

**Must-have:**
- Собственный отдел заселения / качества (не аутсорс)
- Наличие судебных споров с дольщиками (проверить по картотеке арбитража)

**Nice-to-have:**
- Использует CRM для приёмки (Базис, Домиленд) — потенциал интеграции
- Публичные новости о проблемах с качеством / исках

**Disqualifiers:**
- Только элитка (мало квартир, другая динамика споров)
- Банкротство / финансовые проблемы

### Research Tasks

#### Task 1.1: Составить список топ-30 девелоперов с pain signals

Для каждой компании заполнить:

| Поле | Что искать |
|------|------------|
| Название | — |
| Выручка / объём сдачи | ЕРЗ.РФ, Forbes, SPARK |
| # квартир в год | ЕРЗ.РФ |
| Регион | — |
| Pain signals | Судебные иски (kad.arbitr.ru), негативные отзывы (ЕРЗ.РФ, Яндекс), новости о проблемах |
| ЛПР (если найдётся) | HH.ru (вакансии), LinkedIn, пресс-релизы |
| Текущие инструменты | Упоминания в новостях, партнёрства с Базис/Домиленд |

**Минимум:** 20 компаний с заполненными полями, из них 10 — с конкретными pain signals.

#### Task 1.2: Найти судебную статистику по конкретным девелоперам

Источники:
- kad.arbitr.ru — поиск по ИНН/названию, категория «споры из договоров ДДУ»
- sudact.ru — поиск судебных решений
- Правовые базы (Консультант+, Гарант)

**Что нужно:**
- Топ-10 девелоперов по количеству исков от дольщиков за 2023-2025
- Средняя сумма взыскания
- Примеры громких дел (если есть)

#### Task 1.3: Entry Points — Сообщества и события

| Тип | Что найти |
|-----|-----------|
| Конференции | Даты, стоимость участия, кто спикеры (ЛПР девелоперов?) |
| Telegram-каналы | Название, # подписчиков, активность |
| Ассоциации | Членство, как вступить, контакты |
| Интеграторы | Кто внедряет CRM девелоперам (может представить нас) |

**Конкретные события для проверки:**
- «Цифровой девелопмент» (EventToday) — даты 2026?
- Конференции ЕРЗ.РФ
- CFO Russia «Цифровизация в строительном бизнесе» — март 2026?

## Segment 2: Стройконтроль генподрядчиков

### Company Profile

**Кого ищем:**
- Генподрядчики жилого строительства с выручкой >1 млрд ₽/год
- Собственный отдел стройконтроля (от 3 человек)
- 5+ объектов одновременно

**Must-have:**
- Должность «начальник стройконтроля» или аналог (проверить HH.ru)
- Не используют BIM-платформы конкурентов (PlanRadar, Autodesk)

**Nice-to-have:**
- Упоминания о цифровизации стройконтроля в новостях
- Участие в конференциях по стройконтролю

**Disqualifiers:**
- Только коммерческое строительство (другие регуляторные требования)
- Уже используют продвинутые платформы (ПИК — BIMTeam, Самолёт — 10D)

### Research Tasks

#### Task 2.1: Составить список топ-30 генподрядчиков

Для каждой компании заполнить:

| Поле | Что искать |
|------|------------|
| Название | — |
| Выручка | SPARK, Rusprofile, Forbes |
| # объектов | Сайт компании, новости |
| Регион | — |
| Наличие стройконтроля | HH.ru (вакансии «стройконтроль», «инспектор СК») |
| Pain signals | Споры с субподрядчиками (kad.arbitr.ru), задержки сдачи (новости) |
| Текущие инструменты | Упоминания PlanRadar, Excel, собственные системы |
| ЛПР | HH.ru, LinkedIn |

**Исключить из списка:** АПРИ (уже пилот), ПИК, Самолёт (собственные платформы).

**Минимум:** 20 компаний, из них 10 — с вакансиями стройконтроля (= есть отдел).

#### Task 2.2: Pain Signal Search — Где искать

| Источник | Что искать |
|----------|------------|
| HH.ru | Вакансии «инспектор строительного контроля», «начальник стройконтроля» — много вакансий = боль с кадрами |
| kad.arbitr.ru | Споры генподрядчик vs субподрядчик — категория «подряд» |
| Новости | «задержка сдачи [название]», «дефекты [название]», «конфликт с подрядчиком» |
| Telegram | Упоминания компаний в «Цифровое строительство», НОТИМ |

#### Task 2.3: Entry Points — Конференция «Строительный контроль»

**Критически важно:** Конференция skconf.ru (Сочи, февраль 2026) — главный канал.

Нужно узнать:
- Точные даты 2026
- Программа / список спикеров
- Стоимость участия / спонсорства
- Кто из целевых компаний участвовал в 2024-2025 (архив)

**Другие каналы:**
- ЦИПР (Нижний Новгород) — даты 2026?
- MOSТИМ — формат, участники
- Telegram «Цифровое строительство» — кто админы, как разместить контент

## Segment 3: Нефтегаз (Prepared for Fast Follow)

### Company Profile

**Кого ищем (после решения blocker C3):**
- НПЗ вне экосистемы СИБУР (они уже внедрили своё)
- Средние НПЗ (не топ-5, где длинные циклы)
- С обязательными регламентными обходами

**Must-have:**
- Классификация как КИИ (ФЗ-187)
- Нет собственной системы обходов

**Disqualifiers:**
- Входят в СИБУР (7000+ юзеров своей системы)
- Используют VGL Патруль / ОПТИМУМ

### Research Tasks (Lower Priority)

#### Task 3.1: Mapping НПЗ вне крупных экосистем

Список 38 НПЗ России → исключить СИБУР, ЛУКОЙЛ (VGL), Сургутнефтегаз (ОПТИМУМ).

**Потенциальные targets:**
- Независимые НПЗ
- НПЗ Роснефти (если нет централизованной системы)
- Газпром нефть (проверить текущие решения)

#### Task 3.2: Интеграторы нефтегаза

Кто внедряет ИТ-решения на НПЗ:
- КРОК («Цифровой рабочий» — уже работают с нефтегазом)
- АМТ-Груп
- IBS

**Задача:** Понять, можно ли войти через партнёрство с интегратором.
