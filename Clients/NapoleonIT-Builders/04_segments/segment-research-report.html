Логика перехода

Pin 1 → Pin 2: Один девелопер (напр. DOGMA) покупает ТОР для стройконтроля → видит результат → предлагает отделу качества для приёмки. Продажа внутри аккаунта, не cold outreach.
Pin 2 → Pin 3: Референс «крупнейший девелопер использует ТОР» → нефтегаз видит зрелость. Параллельно решается on-prem для КИИ.

Beachhead: Девелоперы — Приёмка квартир 44/50

Критический нюанс C2

73% исков от дольщиков — о нарушении сроков, только 18% — о дефектах качества. Видеофиксация ТОР релевантна для 18%. Адресный размер боли: ~18k исков/год × 850k ₽ = ~15 млрд ₽. Всё ещё огромно, но C2 остаётся [H] до валидации интервью.
