# Project Instructions

## Agent Commands

When user says "run [agent]" — read the agent file from `agents-accessible/` folder and adopt its role/methodology. Do not copy files or execute scripts.

### Pipeline Agents (numbered)
- `01_positioning-agent.md`
- `02_monetization-agent.md`

### Support Agents (in _support/)
- `ai-cleaner-agent.md`
- `positioning-reviewer-agent.md`
- `restructuring-agent.md`

## Project Structure

```
Clients/[ClientName]/
├── _input/           # Raw client materials
├── 01_positioning/   # Positioning outputs
├── 02_monetization/  # Monetization outputs
├── 03_combined/      # Combined docs
├── 04_segments/      # Segment research
├── 05_offers/        # Offer hypotheses
├── 06_entry/         # Entry strategy
└── 07_gtm_final/     # Final GTM deliverable
```
